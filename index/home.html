Cara Menggunakan Aplikasi

1. Login ke Aplikasi

Masukkan username dan password yang telah Anda daftarkan untuk masuk ke aplikasi.

2. Navigasi Menu

Gunakan menu di bagian bawah untuk mengakses berbagai fitur aplikasi:

  • Transaksi: Lakukan simpanan atau pinjaman sesuai kebutuhan Anda.
  • Profil: Untuk melihat latar belakang aplikasi ini.
  • Pengaturan: Digunakan pengguna untuk mengelola informasi pribadi.
  • Logout: Keluar dari akun Anda untuk menjaga keamanan.

3. Melakukan Transaksi

Pilih menu Transaksi, lalu ikuti petunjuk di layar untuk menyelesaikan transaksi simpanan atau pinjaman.

4. Mengelola Profil

Pilih menu Profil untuk memperbarui informasi seperti nama lengkap, email, atau password.

5. Logout

Setelah selesai menggunakan aplikasi, pastikan untuk logout agar akun Anda tetap aman.
